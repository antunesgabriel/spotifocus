# NextJS Scaffold

Este é o template básico que criei para trabalhar em projetos que irei utilizar NextJS

## O que ele possui

- [x] Tailwind
- [x] Typescript
- [x] Setup Jest
- [ ] Setup Cypress (Em breve)

## Como usar

```console
$ yarn create next-app PROJECT_NAME -e https://github.com/antunesgabriel/nextjs-scaffold
```
